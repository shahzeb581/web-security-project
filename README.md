# Advanced Threat Detection & Web Security

## What This Project Does
A Python Flask web application secured with real-world security techniques.

## Security Features
- Brute Force Detection (monitor.py)
- Rate Limiting (Flask-Limiter)
- API Key Authentication
- CORS Configuration
- Security Headers & CSP (Flask-Talisman)

## How to Run
pip install flask-talisman flask-limiter flask-cors
python app.py
